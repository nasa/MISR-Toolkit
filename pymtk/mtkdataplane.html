2.5 MtkDataPlane

class MtkDataPlane
Contains data and map information. 

>>> r = MtkRegion(37, 50, 50)
>>> d = MtkFile('../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf').grid('BlueBand').field('Blue Radiance').read(r)
